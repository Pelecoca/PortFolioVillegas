# Javier Villegas | Portafolio Visual

Este es el repositorio oficial del portafolio de **Javier Villegas**, Tecnólogo en Desarrollo de Medios Gráficos Visuales. El proyecto muestra una selección de trabajos en branding, producción audiovisual, diseño para redes sociales e innovación con Inteligencia Artificial.

## 🚀 Características

- **Diseño Responsivo**: Optimizado para dispositivos móviles y escritorio.
- **Interfaz Fluida**: Animaciones con Three.js y transiciones suaves.
- **Portafolio Dinámico**: Filtrado de proyectos por categoría (Branding, Video/3D, Social Media).
- **Contenido Multimedia**: Galería integrada con soporte para imágenes de alta resolución y videos.

## 📂 Estructura del Proyecto

El proyecto está organizado de la siguiente manera para facilitar su mantenimiento y escalabilidad:

```text
/
├── assets/
│   ├── docs/       # Documentos oficiales (Hoja de Vida, etc.)
│   ├── images/     # Recursos de imagen (PNG, JPG)
│   └── videos/     # Recursos de video (MP4)
├── css/            # Hojas de estilo (styles.css)
├── js/             # Lógica de programación (script.js)
└── index.html      # Punto de entrada principal
```

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (ES6+).
- **Gráficos 3D**: [Three.js](https://threejs.org/) para el fondo animado.
- **Iconografía**: Font Awesome 6.
- **Tipografía**: Google Fonts (Outfit).
